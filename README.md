Objetivo
  O objetivo deste projeto é desenvolver e simular um sistema de Semáforo Inteligente capaz de controlar o tráfego de veículos de forma automática e segura, ajustando os tempos dos sinais conforme o fluxo de veículos e condições especiais,
como chuva e falhas de sensores. O projeto visa aplicar conceitos de Internet das Coisas (IoT), redes, sistemas operacionais e segurança da informação, proporcionando uma solução eficiente e confiável para o controle do tráfego urbano.
